# YTLoader

無須 Youtube Preminum 即可免費下載影片
繁體中文介面一目瞭然
簡單多功能的實用影片下載器

# 功能

**影片畫質調整**
目前支援:
- 最高畫質 (預設)
- 1080p
- 720p
- 360p
- 最低畫質

**音訊/影片選擇**

- MP4 影像 (.mp4)
- MP3 音訊 (.mp3)

# 需求

需安裝 Python 3 或更高版本
